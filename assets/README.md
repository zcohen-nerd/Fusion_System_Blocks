# Media Assets for Fusion System Blocks

This directory contains media assets for documentation and public presentation (screenshots, demos, logos, diagrams).

## File Naming Conventions

- **Screenshots:** `screenshot_[feature]_[version].png`
- **Demos:** `demo_[feature]_[version].gif` or `.mp4`
- **Logos:** `logo_[variant]_[size].png`
- **Diagrams:** `diagram_[type]_[version].png`

## Guidelines

- Use PNG for UI screenshots, GIF/MP4 for demos, SVG for scalable graphics.
- Optimize file sizes for web viewing.
- Follow Fusion 360 color scheme and styling for consistency.
